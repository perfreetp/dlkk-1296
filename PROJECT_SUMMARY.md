# 简历优化助手 - 项目总结

## 项目概述

**简历优化助手** 是一款面向应届生的在线简历优化工具，帮助用户在投递前快速打磨简历，提升求职竞争力。

## 核心功能

### ✅ 已实现功能

1. **简历导入页**
   - 支持 PDF 文件上传（使用 pdf.js 解析）
   - 支持粘贴文本导入
   - 实时预览简历内容
   - 自动解析简历结构（基本信息、教育背景、工作经历、项目经历、技能）

2. **职位匹配页**
   - 输入目标岗位描述
   - 智能提取关键词（技术栈、软技能）
   - 计算简历与岗位的匹配度百分比
   - 识别缺失的关键词
   - 生成个性化优化建议

3. **优化编辑页**
   - 实时编辑简历各部分内容
   - 空泛表达自动检测与标记
   - 量化改写建议生成
   - 版本保存与管理
   - PDF 导出功能

4. **版本对比页**
   - 并排对比不同版本
   - 差异高亮显示（新增/删除/修改）
   - 差异统计面板
   - 逐条选择应用优化

5. **投递记录页**
   - 记录投递公司和职位
   - 状态管理（待处理/面试中/Offer/拒绝/无反馈）
   - 投递统计分析
   - 简历版本关联
   - 备注功能

6. **素材库页**
   - 项目经历素材管理
   - 技术栈和成果指标
   - 标签分类
   - 搜索和筛选

7. **设置页**
   - 默认简历设置
   - 数据导入导出（JSON 格式）
   - 数据统计
   - 清除所有数据

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **路由管理**: React Router v6
- **状态管理**: Zustand
- **PDF 处理**: pdf.js
- **图标**: Lucide React
- **数据持久化**: LocalStorage

## 核心算法

### 1. 关键词提取
- 文本预处理（去除标点、停用词）
- 词频统计
- TF-IDF 计算
- 分类整理（技术栈、软技能、职责）

### 2. 空泛表达检测
- 模式匹配（常见空泛短语）
- 缺少量化数据检测
- 主观表述识别

### 3. 量化改写建议
- 上下文感知建议生成
- 指标模板匹配
- 动词优化建议

### 功能使用流程

1. **导入简历**
   - 进入首页选择上传方式
   - PDF 或粘贴文本
   - 查看解析结果

2. **职位匹配分析**
   - 输入目标岗位描述
   - 获取匹配度评分
   - 查看缺失关键词和优化建议

3. **优化编辑**
   - 根据建议修改简历
   - 保存优化版本
   - 导出 PDF

4. **版本对比**
   - 选择要对比的版本
   - 查看差异统计
   - 选择性应用优化

5. **投递记录**
   - 添加投递公司信息
   - 跟踪面试进度
   - 查看统计分析

## 数据安全

- 所有数据仅存储在浏览器本地
- 不上传到任何服务器
- 支持数据导出备份
- 支持数据导入恢复
